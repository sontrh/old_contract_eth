# Ethereum Early Contracts Archive (2015–2016)

This repository contains a collection of smart contracts deployed during Ethereum's early years (2015–2016).

Many of these contracts have remained untouched for years and were created before modern smart contract security practices became standard. As a result, some contracts may contain forgotten assets, misconfigurations, or vulnerabilities that were never discovered.

## Purpose

This repository is intended for:

* Smart contract researchers
* Security auditors
* Bug hunters
* Blockchain historians
* Ethereum enthusiasts

Think of it as a blockchain treasure hunt through Ethereum's earliest contracts.
